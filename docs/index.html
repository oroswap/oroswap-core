Welcome to Oroswap

Oroswap is a decentralized exchange (DEX) built on the Zigchain blockchain using CosmWasm smart contracts.

Quick Start:
  • Deploy contracts
  • Execute transactions
  • Monitor events

Core Components

  • Factory Contract: Manages pair creation and routing
  • Pair Contract: Handles liquidity pools and swaps
  • Router Contract: Provides swap interface
  • Coin Registry: Manages token metadata
  • Incentives: Handles liquidity mining rewards
